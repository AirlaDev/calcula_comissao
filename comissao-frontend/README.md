# Sistema de Cálculo de Comissão - Frontend

Frontend desenvolvido com Next.js, React e TypeScript.

## Instalação

```bash
npm install
```

## Execução

```bash
npm run dev
```

O sistema estará disponível em `http://localhost:3000`

## Estrutura

- `app/page.tsx` - Página principal com interface para calcular comissões
- `app/layout.tsx` - Layout base da aplicação
- `app/globals.css` - Estilos globais
- `app/page.module.css` - Estilos da página principal
